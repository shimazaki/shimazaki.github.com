計算機関係

昨年度まではLinuxもWindowsもスタンドアローンで使用していました（一部
Sambaでファイルの共有をしていました）。
学生はWindowsを主に使っていたようです。
本年度はXRSを全員が使用することになるようですから、Linuxの使用が主になる
と思います。
そこでLinuxの環境を整備しました。

５２３のパソコンの名前は向かって左から、sparrow, swallow, penguin, hatoとなっています。
penguin以外はWindowsとLinuxのデュアルブートになっています。各計算機のOSの構成は次のようになっています。

sparrow Windows XP/ Linux (debian, sarge)
swallow Windows XP/ Linux (debian, sarge)
penguin Linux (debian, sarge)
hato Windows XP/ Linux (debian, sarge)

古いRed Hat Linuxを一掃し、5台すべてにDebian GNU/Linuxを導入しました。
hatoにNISとNFSを導入し、アカウントとディスクを一元管理します。hatoはNIS,NFS,Sambaサーバーとして常時Linuxを稼働させます。
データは定期的にバックアップをとるようにします。
また現時点ではhatoのDebianでのみXRSをコンパイルすることができます。

NISとNFSの設定は終わり、基本的な使用はすでにできるようになっています。微調整などは初回の授業（来週の水曜日）までに完了させます。XRSのインストールに時間がかかることも予想されますので、コンパイルはhatoで行うようお願いします。実行は他のマシンでも行えます。
