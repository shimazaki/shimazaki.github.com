Online Bin Size Selection (ver. 0.0 beta)

This web application calculates an optimal bin size for the histogram of the data you post.
For the details of the method, please refer to
Shimazaki H. and Shinomoto S., A method for selecting the bin size of a time histogram Neural Computation [pdf]
